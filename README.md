# AllieDEals - PHP/MySQL E-Commerce Website

Welcome to **AllieDEals**, a fully functional e-commerce platform built with PHP and MySQL for XAMPP.

## Quick Start

1. **Import Database**: Import `database_setup.sql` in phpMyAdmin
2. **Update Config**: Edit `config/config.php` with your Razorpay keys
3. **Install Razorpay SDK**: Run `composer require razorpay/razorpay` OR download manually
4. **Access**: Open http://localhost/alliedeals.com/

## Features

✅ Product catalog with variants (size/color)  
✅ Shopping cart (session-based)  
✅ User authentication  
✅ Razorpay payment integration  
✅ Order management  
✅ Responsive design (Tailwind CSS)
